# TascaS2.04.Mongodb_queries

## 📌 Introduction  
This project focuses on learning how to work with **MongoDB queries**, exploring the fundamentals of querying data in a **NoSQL environment**.  

The database used in this project is **restaurants**, which contains information about restaurants. The goal is to practice retrieving and manipulating information using MongoDB queries.  

By working with these queries, you will:  
- Learn how to extract data from a NoSQL database.  
- Understand how to filter, sort, and aggregate information.  
- Gain hands-on experience executing queries in **MongoDB Shell**.  

## 🔧 Getting Started  
To run this project locally:  

```bash
# 1. Clone the repository
git clone https://github.com/pantalois/TascaS2.04.Mongodb_queries.git

# 2. Import the JSON file into your MongoDB instance
Use the JSON file restaurants.json

# 3. Open MongoDB Shell
Navigate to the database restaurants and execute the queries from the queries.JSON file in the Mongodb Shell
You can run them one by one and explore the results
